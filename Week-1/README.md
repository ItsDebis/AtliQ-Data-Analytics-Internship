# Week 1 - Data Cleaning & HR Analysis

## Overview
Worked on cleaning and analyzing employee attendance data to generate meaningful insights.

---

## Tasks Performed
- Cleaned raw attendance dataset by standardizing date formats and employee IDs  
- Removed inconsistencies and duplicate records to improve data quality  
- Transformed categorical data into standardized formats (WFO, WFH, etc.)  

---

## Analysis Performed
- Analyzed employee attendance patterns and work-from-home trends  
- Identified key insights such as WFH percentage and attendance distribution across days  
- Evaluated employee participation and behavior trends  

---

## Tools Used
- Excel / Power BI  

---

## Key Learnings
- Data cleaning and preprocessing techniques  
- Importance of structured data for analysis  
- Converting raw data into actionable insights
